# Annotated CSS Template

This folder contains the reusable editorial CSS system used for Annotated.

Use `annotated-editorial-template.css` as the baseline for new pages, embeds, or companion surfaces that should match the current product style:

- warm paper background
- Newsreader serif headlines and commentary
- Inter UI chrome
- ruled lines instead of cards
- quiet metadata
- restrained indigo links and actions
- no shadows, dark backgrounds, gradients, or CSS frameworks

Recommended font include:

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&family=Newsreader:ital,wght@0,400;0,600;0,700;1,400&display=swap" rel="stylesheet">
```
